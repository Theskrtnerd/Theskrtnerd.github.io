3:T21a7,<h2>Introduction</h2>
<p>During the tough days of Covid, stuck with online studies and feeling down, Discord and my friends ✨ were a lifeline. Chatting and playing games on Discord made things a lot more fun.</p>
<p>I stumbled upon Discord bots, especially the Game Discord Bots, and thought it'd be cool to play soccer games. But, there weren't any. After some Google searching, I discovered I could make my own Discord Bot using Python.</p>
<p>With just one year of Python experience, I learned to code and set it up on repl.it. I kept adding our favorite games like "Guess the player," "Missing 11," and "Soccer Quiz."</p>
<p>This project is my favorite. It brought me joy and helped my friends and me get through one of the toughest times for everyone.</p>
<h2>Process</h2>
<p>Let's walk through the process of creating the bot:</p>
<h3>1. Register the bot</h3>
<p>Start by visiting the <a href="https://discord.com/developers/applications">Discord Developer Portal</a>, selecting "New Application," and filling in the necessary information for your Discord bot. Create a new Bot Token, a crucial element for bot development.</p>
<p><img src="/soccer-discord-bot-in-covid-days/1705682228710.png" alt="1705682228710"></p>
<p><strong>Note:</strong> If you've already created the token, opt for "Reset Token" to generate a new one.</p>
<h3>2. Invite the bot to your server</h3>
<p>Now, invite the bot to your server for testing. Navigate to <code>OAuth2 -> URL Generator</code>, choose the options <code>bot</code> and <code>Administrator</code> for your local server, and generate a URL like <code>https://discord.com/api/oauth2/authorize?client_id={client_id}&#x26;permissions=2048&#x26;scope=bot.</code> Paste this URL into your browser's search bar to add the bot to your server.</p>
<p><img src="/soccer-discord-bot-in-covid-days/1705683032435.png" alt="1705683032435"></p>
<p>Once you see the bot's incoming message in your server, it confirms successful addition.</p>
<h3>3. Start coding ✨</h3>
<p>Now, to start, I would recommend trying to run your code on <a href="https://replit.com">repl.it</a>. Create an account if you haven't already, and create a new project.</p>
<p>Create 2 files <code>main.py</code> and <code>keep_alive.py</code>. These are the only 2 files you need right now.</p>
<p>For <code>keep_alive.py</code>, paste the following code in:</p>

<h3>4. Add your own touch</h3>
<p>With all of that setup, you can now create anything you want! Just think of something you can do with the Discord Bot and make it yourself.</p>
<p>For reference, I will talk about the process of adding the Soccer Games. So in order to create the soccer games like starting 11 or guess the player, I need to get data from previous matches, or player's history.</p>
<p>But to do that, I need to find some kinds of API, or I have to crawl data from a soccer site. After looking around, I cannot find a free API that has things I need. So I decided to crawl data from <a href="https://www.transfermarkt.com">Transfermarkt</a>. This site has everything I need, from player's transfer history, to the line-ups of all the matches in history.</p>
<p>So firstly, I used <code>bs4</code> to crawl data. But a downside of <code>bs4</code> is that it only works on static web. And because transfrmarkt is a dynamic generated website, this library didn't work. Luckily, I found another alternative, which is <code>requests-html</code>. This works simillar to <code>bs4</code>, but it will wait for the website to load before crawling the data.</p>
<p>Below is a code demo of the functions for my games, which contains all the data crawling and game creating.</p>
